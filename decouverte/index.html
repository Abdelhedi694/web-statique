Au cours de cette formation nous allons voir 2 matières :

L'intégration

Le HTML :

Le HTML sert à créer le contenu du site : elle va contenir uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fond, etc.

Le développement

Le Javascript :

Javascript est un langage de programmation de script employé pour rendre une page web interractive, le comportement de la page et son apparence pourra varier en fonction de certains paramètre.

Formation intensive :

Le programme de formation est découpé en différents modules.
